# 2026-03-13_04_MVP13_Codex.md

## 目的
COBOL → C# 変換プロジェクトにおいて **MVP13: WRITE（SEQUENTIAL出力）** を実装する。
MVP12（READ）の次の段階として、逐次出力の最小基盤を構築する。

このMVPのゴールは

- COBOLの `WRITE` をC#へ変換できる
- sequential output ができる
- 自動テストで動作を保証できる

ことである。

---

# 実装対象

対応するCOBOL構文

- SELECT
- FD
- OPEN OUTPUT
- WRITE record-name
- CLOSE

対象は **SEQUENTIAL OUTPUT のみ**

---

# 非対応（今回は実装しない）

以下は未対応とする。入力された場合は検出できるようにする。

- WRITE ... FROM
- WRITE AFTER ADVANCING
- WRITE BEFORE ADVANCING
- printer control
- indexed file write
- rewrite

未対応は `NotSupportedException` などで明示する。

---

# 実装方針

## 1 ランタイム

逐次出力用の最小ランタイムを追加する。

責務

- ファイル作成
- 1行書き込み
- flush
- close

参考構造

```csharp
public sealed class SequentialFileWriter : IDisposable
{
    private readonly StreamWriter _writer;

    public SequentialFileWriter(string path)
    {
        _writer = new StreamWriter(path);
    }

    public void WriteLine(string record)
    {
        _writer.WriteLine(record);
    }

    public void Dispose()
    {
        _writer.Dispose();
    }
}
```

※既存runtime設計に合わせて調整すること。
